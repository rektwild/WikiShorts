// App Name
WikiFlick

// Subtitle
日知其所亡

// Promotional Text
每日為君精擇維基大典之奇文！釋君之惑，而學新知。

// Description
此致知之樂道也！WikiFlick以當世之形，速覽之便，呈維基大典之無窮學海於指掌之間。免乎長篇之繁，雜說之亂，日日發見新奇之事。

WikiFlick之設，意在使學為日用之常。無論君有五分之暇，或在行旅之中，皆可為增廣見聞之良佐。

其要有三：

• 今日之選：編輯精擇，奇文佳篇，萬勿錯過。
• 偶然之得：一觸隨機，天降奇文，以廣見聞。
• 智能之尋：心中所惑，即刻可索，相關之文，應念而至。
• 心愛之藏：心儀之文，或欲後讀者，皆可輕易存之，以建爾雅之書齋。

何用WikiFlick？

蓋因學不必為苦事也。通覽萬象，由格致、藝術，至史地、科技，無所不包。願與君偕，日進一智。

即刻下載，開啟探索之旅！
