
// Mongolian (Mongolia) - mn

// App Name
WikiShorts: Шинэ Мэдлэг Нээ

// Subtitle
Өдөр Бүтэн Шинэ Мэдлэг Нээ

// Promotional Text
Бид танд зориулж Википедиагийн хамгийн сонирхолтой нийтлэлүүдийг өдөр бүр сонгодог! Сониуч зангаа дарж, шинэ зүйл сураарай.

// Description
Мэдлэг олж авах хамгийн таатай аргатай танилцана уу! WikiShorts нь Википедиагийн хязгааргүй ертөнцийг орчин үеийн, хурдан хэлбэрээр таны хурууны үзүүрт авчирдаг. Урт, төвөгтэй нийтлэлүүдэд төөрөхийн оронд өдөр бүр шинэ, гайхалтай баримтуудыг нээгээрэй.

WikiShorts нь суралцахыг өдөр тутмын дадал болгох зорилготой юм. Та таван минутын завсарлага авсан ч, аялаж яваа ч бай, энэ нь таны ерөнхий мэдлэгийг тэлэхэд төгс хамтрагч болно.

ГОЛ ОНЦЛОГ:

• Өдрийн Сонголт: Манай редакторуудын анхааралтай сонгосон хамгийн сонирхолтой сэдвүүд болон "өдрийн нийтлэлийг" бүү алдаарай.
• Санамсаргүй Нээлт: Товчлуур дарж санамсаргүй нийтлэлээр өөрийгөө гайхшруулж, мэдлэгийн хүрээгээ тэлээрэй.
• Ухаалаг Хайлт: Сонирхож буй аливаа сэдэвт шууд хандаж, холбогдох нийтлэлүүдийг хурдан олоорой.
• Дуртай Зүйлсийн Жагсаалт: Хувийн номын сангаа бүрдүүлэхийн тулд дуртай нийтлэлүүдээ эсвэл дараа уншихыг хүссэн зүйлсээ хялбархан хадгалаарай.

Яагаад WikiShorts гэж?

Яагаад гэвэл суралцах нь уйтгартай байх албагүй. Шинжлэх ухаан, урлагаас эхлээд түүх, технологи хүртэлх мянга мянган сэдвээр товч мэдлэгийг аваарай. WikiShorts-тэй хамт өдөр бүрийг арай илүү ухаалаг болго.

Одоо татаж аваад, нээлт хийж эхлээрэй!
