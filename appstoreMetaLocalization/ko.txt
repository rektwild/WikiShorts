// App Name
WikiFlick

// Subtitle
매일 새로운 지식 발견

// Promotional Text
매일 가장 흥미로운 위키백과 문서를 선정해 드립니다! 호기심을 채우고 새로운 것을 배워보세요.

// Description
지식을 얻는 가장 즐거운 방법을 만나보세요! WikiFlick은 위키백과의 방대한 세계를 현대적이고 빠른 형식으로 여러분의 손끝에 제공합니다. 길고 복잡한 문서 속에서 길을 잃는 대신, 매일 새롭고 놀라운 사실들을 발견하세요.

WikiFlick은 학습을 일상의 습관으로 만들도록 설계되었습니다. 5분의 짧은 휴식 시간이든 이동 중이든, 여러분의 일반 상식을 넓히는 완벽한 동반자가 될 것입니다.

주요 기능:

• 오늘의 추천: 저희 에디터들이 엄선한 가장 흥미로운 주제와 '오늘의 문서'를 놓치지 마세요.
• 랜덤 발견: 버튼 하나만 누르면 나타나는 랜덤 문서로 새로운 놀라움을 경험하고 시야를 넓혀보세요.
• 스마트 검색: 궁금한 주제는 무엇이든 즉시 검색하고 관련 문서를 빠르게 찾아보세요.
• 즐겨찾기 목록: 좋아하는 문서나 나중에 읽고 싶은 문서를 쉽게 저장하여 자신만의 개인 라이브러리를 만드세요.

왜 WikiFlick인가?

학습은 더 이상 지루할 필요가 없습니다. 과학과 예술부터 역사와 기술에 이르기까지 수천 가지 주제에 대한 핵심 지식을 얻으세요. WikiFlick과 함께 매일을 조금 더 똑똑하게 만드세요.

지금 다운로드하고 탐험을 시작하세요!
